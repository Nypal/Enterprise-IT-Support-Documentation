# Enterprise IT Support Documentation

## Overview

This repository documents my technical support work, troubleshooting process, ticketing habits, and Windows IT support skills developed through front-line support responsibilities.

The goal is to show how I handle user assistance, Windows 10 and Windows 11 support, network connectivity issues, access control problems, documentation, escalation, and technical communication in a professional environment.

## Professional Context

In my role as a Help Desk and Technical Support Specialist, I support residents, guests, vendors, and internal users by helping resolve common technical and operational issues.

This documentation connects my daily support responsibilities with the technical concepts used in IT Support, Help Desk, Systems Administration, and Operations Technology roles.

## Areas Documented

### Windows User Support

Documentation of common Windows 10 and Windows 11 support tasks, including login issues, basic troubleshooting, system settings, device problems, and user assistance.

### Network Connectivity Troubleshooting

Documentation of troubleshooting steps for internet connection problems, Wi-Fi issues, IP configuration, DNS checks, gateway checks, and escalation when needed.

### Access Control Support

Documentation of support workflows related to user access, access control systems, permission issues, vendor access, and resident service requests.

### Ticketing and Incident Documentation

Documentation of how support requests are recorded, prioritized, updated, resolved, and communicated during shift handoff.

### Printer and Device Support

Documentation of common printer, workstation, peripheral, and device troubleshooting steps.

### Customer Service and Technical Communication

Documentation of how technical problems are explained clearly to non-technical users while maintaining professionalism during high-pressure situations.
